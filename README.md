# Qiuck start
- clone this repo
- set `.env.template` to `.env` and fill in your CLAUDE_API_KEY and OPENAI_API_KEY. The OPENAI_API_KEY is used to get fix json and similarity.
- install requirements `pip install anthropic`
- run `python -m autogpt`
